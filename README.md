## 🛠️ Instalación y Uso

Este es un sitio web estático. Para verlo localmente:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/DiRoCoDEV/dirocodev.github.io.git
   cd dirocodev.github.io
   ```

2. Abre `index.html` en tu navegador web.

Para desplegar en GitHub Pages, sube los archivos a tu repositorio y habilita Pages.
